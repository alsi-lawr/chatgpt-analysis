## [0.1.0] - 2026-07-13

### Added

- Deterministic normalization for supported ChatGPT export ZIP, JSON, JSONL, mapping, and message-list inputs.
- Rebuildable inventory, metrics, bounded segmentation, taxonomy cards, Markdown/JSON reports, and SQLite/FTS indexing.
- Source snapshots, configuration digests, evidence links, and acceptance checks for provenance and consistency.
- Provider-neutral JSON worker queues with retry, quarantine, independent review, and disagreement adjudication.
- Synthetic end-to-end fixtures and tests that require neither a real export nor network access.
- Research software metadata, contribution and security guidance, installed-package smoke checks, and continuous integration.
